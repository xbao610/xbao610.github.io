推荐使用的软件清单-Mac版

2019-06-13

office三件套 尽管有苹果三件套，现在又有了WPS-mac版，用来用去，还是office三件套好用，特别是最新的office2019。 费用：付费使用office365也不是很贵，或者淘宝买永久激活码。 下载地址：https://products.office.com/zh-CN/compare-all-microsoft-office-products?tab=1 或App Store

SnailGit git版本控制客户端 国人开发的git版本控制客户端，可以配合“访达”文件管理器使用，支持文件状态角标显示与右键快捷操作。 费用：免费版支持1个仓库，付费版无限制。 下载地址：https://langui.net/ 或App Store

SnailSVN svn版本控制客户端 与上面的git一样 下载地址：https://langui.net/ 或App Store

Xnip 截长图 清爽简单的截图工具，支持滚动截长图。 费用：免费版有LOGO水印，付费很人性化12元每年。 下载地址：https://zh.xnipapp.com/ 或App Store

Sip 取色器 简单清爽的取色工具，放大镜界面取色超级准确。 费用：免费 下载地址：http://sipapp.io/

FileZilla FTP工具 老牌的FTP上传工具，支持SFTP的22端口，支持在线编辑保存 费用：免费 下载地址：https://www.filezilla.cn/download/client 或App Store

Chrome 谷歌浏览器 最好用的浏览器，开发必备。 费用：免费 下载地址： https://www.google.cn/intl/zh-CN/chrome/ (居然可以打开)

Lantern 蓝灯翻墙客户端 还算稳定的翻墙软件，不定时更新。但是最近也不好用了。 费用：流量付费 下载地址：https://github.com/getlantern/download

Microsoft Remote Desktop 远程控制windows服务器 微软出品的远程桌面连接工具，相当于windows的“远程桌面连接”，默认支持3389端口，不像其他的软件需要在服务器安装服务端。 费用：免费 下载地址：https://docs.microsoft.com/zh-cn/windows-server/remote/remote-desktop-services/clients/remote-desktop-clients 或 App Store 切换到美国地区下载。(奇葩的微软)

Sublime Text 文本编辑器 老牌经典好用的文本编辑器 费用：免费 下载地址：http://www.sublimetext.com/3

MAMP PRO PHP环境工具 功能强大的PHP环境配置工具，支持多版本切换。虽然mac系统自带了php环境，但是折腾起来也挺烦的，如果不想折腾环境可以直接用这个。 费用：基础版免费，高级版付费 下载地址：https://www.mamp.info/en/mamp-pro/

Gridea 静态博客 利用github page或者codeing page 搭建一个静态博客，简单好用。 费用：免费 下载地址：https://gridea.dev/

processon 在线流程图脑图工具 很好用的流程图SaaS工具，Visio的替代品，可惜没有客户端，而且名字也不好记，每次想用还得现百度一下。 费用：部分文件数免费，团队版付费 在线地址：https://www.processon.com/

新媒体管家 公众号编辑器浏览器插件 用了很多公众号编辑器，还是这个好用。支持一些微信公众号技巧的小功能特别好用，比如：文章素材永久链接，错别字检测等等。 费用：免费 下载地址：http://xmt.cn/index

PostMan 接口调试浏览器插件 常用的接口调试浏览器插件，不过觉得新版没有之前的老版好用 费用：免费 下载地址：https://www.getpostman.com/
